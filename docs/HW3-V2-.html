#4.What are the two main differences between a tibble and a data.frame?

First, a tibble prints more nicely. It only shows the first few rows and columns that fit on the screen, while a data.frame may print a lot of issues that may not be necessary

Second, a tibble is stricter and more predictable. For example, it does not automatically change variable types or names as much as a traditional data.frame may.
